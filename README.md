# Financial Sentiment Analysis for Trading Signals

A machine learning project that analyzes financial news sentiment to generate trading signals.

## Project Structure

```
financial-sentiment-analysis/
├── data/                   # All datasets
├── notebooks/             # Jupyter notebooks for exploration
├── src/                   # Production source code
├── tests/                 # Test files
├── models/               # Saved models
├── configs/              # Configuration files
├── reports/              # Generated reports and visualizations
├── deployment/           # Docker and deployment configs
├── scripts/              # Utility scripts
└── docs/                 # Documentation
```

## Getting Started

1. Create virtual environment: `python -m venv venv`
2. Activate it: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
3. Install dependencies: `pip install -r requirements.txt`
4. Start exploring in `notebooks/exploratory/`
